Inputs HTML5

Para usar el color: input type color

Para usar la fecha: input type date

Datetime no sé si funcionaba?

Para poner fecha y hora: input type datetime-local

Para el email: input type email

Para el mes únicamente: input type month

Para el numero: input type number. Puedes manipular el máximo (max), el mínimo (min) y el numero de pasos (step)

Para rango usamos input type range. Puedes manipular el número de opciones del rango (min) y el número de pasos (step)

Para teléfono: input type tel

Para la hora: input type time

Para la url: input type url

Para una única semana: input type week
